Пояснительная записка к проекту.

Для разработки использовались: Python3, Pyqt5, прочие библиотеки python, sqlite studio.

Краткое описание: Стратегия для двух человек
Название: Обычная стратегия

Функционал:
 Главное окно:
  Виджет библиотеки PyQt QMainWindow, на нем расположены части интерфейса, реализованные другими виджетами.
  Функционал:
    В центре карта 8 х 8, на карте происходят основные действия программы, кликнув на любую клетку карты можно получить ее описание
    и возможность сделать какое-то действие во всплывающем окне. Есть четыре типа клеток: трясина, замок, гора, пустая клетка.
     Трясина: Крайне плохая клетка, создана, для того чтобы помешать игроку. Любой герой игрока может в ней потонуть(навсегда)
     Замок: для победы нужно снять все очки здоровья замку, каждый удар по нему сносит 100 очков, в то время как всего их 500
     Гора: обьект, не приносящий никакой пользы игроку, на него нельзя встать, через него нельзя пройти.
     Пустая клетка: Пустая клетка, на нее можер встать, на ней можно сражаться
  Герои: у каждого игрока их 3. Сделаны с помощью базы данных и идалоговых окон, показатели - здоровье и сила. для каждлого вида героя есть 3 навыка
     Героями совершаются основные действия. За свой ход игрок модет переместить одного из своих героев / атаковать вражеского героя одним из своих
     При поражении всех 3-х героев игра не закончится - нужно будет дойти до замка и снять хдоровье ему.
  Ход: только герои могут передвигаться по карте. для того, чтобы передвинуть героя, нужно кликнуть на него, а затем на клетку, в которую вы хотите
      его передвинуть. На это, естественно, есть ограничения ввиде максимальной дистанции, для каждого класса она своя.
  Бой:
    Сам бой также реализован с помощью базы данных и диалоговых окон. Для того, чтобы атаковать, нужно сначала кликнуть по герою, которым
    вы хотите атаковать, а затем на того, кого вы хотите атаковать. Вы увидите диалоговое окно с умениями. Всего в игре 9 умений, но новые
    можно свободно добавить через базу данных. 3 умения для каждого класса. Всю информацию об умении можно получить, введя его название в ввод лога
    реализован textedit-ом внизу матрицы и нажав кнопку 'go', если вражеский герой умирает, или происходит что-то интересное эта информация 
    будет выведена в лог.
  Конец: Игра заканчивается, когда один из игроков добил вражеский замок. Также возможна и ничья, поскольку некоторые способности героев способны уничтожить
         и их самих. В случае ничьей ничего не произойдет, поскольку разработчик очень ленив.
      
 Внизу расположена информация о игроке,кнопка смены хода.
